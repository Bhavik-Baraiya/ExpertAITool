📘 NLCEX Expert AI – POC App

A lightweight Proof of Concept (POC) iOS application designed to assist NCLEX students with Apple Intelligence, AI-driven support. The app provides concise answer summaries, clear rationale explanations, and practical NCLEX tips based on user-submitted questions.
Built using Apple’s native SwiftUI and FoundationModel framework, this app demonstrates how on-device AI can be leveraged to process and respond to NCLEX-related queries efficiently and contextually.

🚀 Key Features
    • 🧠 AI-generated answer summaries for NCLEX questions
    • 📖 Clear and structured rationale explanations
    • 💡 Actionable NCLEX tips to improve exam readiness
    • ⚡ Lightweight and fast POC implementation
    • 🍏 Powered by Apple’s FoundationModel for query processing

🎯 Purpose
This project serves as an experimental prototype to explore the feasibility of integrating Apple’s AI capabilities into educational tools for nursing exam preparation.
